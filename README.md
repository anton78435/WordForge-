📊 WordForge — Анализатор частоты слов на 7 языках
WordForge — коллекция из семи независимых реализаций анализатора частоты слов в текстах. Каждая версия работает на своём языке программирования и предлагает расширенные возможности для обработки текста, подсчёта статистики и визуализации результатов.

✨ Уникальные возможности (общие для всех версий)
📝 Анализ частоты слов в текстовых файлах или введённом тексте.

🔍 Игнорирование стоп-слов (предлоги, союзы, местоимения) для более точного анализа.

📊 Визуализация результатов в виде ASCII-гистограммы (в CLI) или таблицы с сортировкой.

🔤 Приведение к нижнему регистру и удаление пунктуации.

📈 Статистика: общее количество слов, уникальных слов, топ-N самых частотных слов.

💾 Экспорт результатов в CSV и JSON.

🌍 Поддержка нескольких языков (русский, английский — настраиваемые стоп-слова).

🧹 Очистка текста — удаление HTML-тегов, чисел, специальных символов.

🖥️ Интерфейсы:

Командная строка (CLI) с аргументами.

Интерактивный режим (ввод текста или пути к файлу).

Веб-интерфейс в PHP и JavaScript.
